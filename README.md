# 6713

A sovereign application with a clean slate.

## Features

- **Navigation Bar**: Fixed at the top with glassmorphism effect
- **Tabs**: Hue, Wall, Live, $$$4U
- **Upload Button**: Centered '+' button for photo/video gallery access (only visible for verified users)
- **Theme**: Pure black (#000) background with white text

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
